# 広告除外依頼 文章生成ツール

リスティング広告の除外依頼文章をAIで簡単に作成できるツールです。

## 特徴

- 🚀 **簡単操作**: 企業名とキーワードを入力するだけで依頼文章を生成
- 🎯 **お試し機能**: 初めての方でもサンプルデータで動作確認可能
- 🤖 **AI生成**: OpenAI APIを使用して適切な依頼文章を自動生成
- 🎨 **ビジネス風デザイン**: 親しみやすいUI/UX
- 📋 **ワンクリックコピー**: 生成された文章を簡単にコピー
- 🔄 **再生成機能**: 異なるニュアンスの文章を生成可能
- 🎭 **トーン選択**: 丁寧度やスタイルを選択可能

## 使い方

### 1. ファイルを開く

`ad-exclusion-request.html` をブラウザで開くだけです！
- **ダブルクリック**: `ad-exclusion-request.html` をダブルクリックしてブラウザで開く
- **ドラッグ&ドロップ**: `ad-exclusion-request.html` をブラウザにドラッグ&ドロップ

### 2. 情報を入力

1. **あなたの会社名**: 依頼を送る側の会社名
2. **あなたの名前**: 依頼者の名前
3. **あなたのメールアドレス**: 連絡先メールアドレス
4. **対象企業名**: 広告を出している企業名
5. **除外したいキーワード**: 1行に1つずつキーワードを入力
6. **文章のトーン**: 丁寧度やスタイルを選択（標準・非常に丁寧・ビジネス風・フレンドリー）

### 3. 文章を生成

「✨ 依頼文章を生成する」ボタンをクリックすると、AIが適切な依頼文章を作成します。

### 4. コピーして送信

生成された文章をコピーして、対象企業のフォームまたはメールに貼り付けて送信してください。

## 生成される文章の構成

生成される依頼文章には以下の要素が含まれます：

1. **宛先**: マーケティングご担当者様
2. **挨拶**: フォームからの連絡であることを記載
3. **自己紹介**: 会社名、名前、メールアドレス
4. **依頼の説明**: 
   - リスティング広告の除外依頼について
   - 現状の説明（検索結果で広告が表示されている）
   - 依頼の理由（意図しない配信、ユーザーとの適切な接点）
   - 除外設定の検討依頼
5. **該当メディア**: Google、Yahoo!、Microsoft
6. **除外キーワードリスト**: 入力したキーワード一覧
7. **マッチタイプ**: フレーズ一致

## AI機能について

### バックエンドAPI

- **OpenAI API統合**: Vercel Serverless FunctionでOpenAI APIを呼び出します
- **APIキー管理**: Vercelの環境変数でAPIキーを管理
- **フォールバック機能**: バックエンドが利用できない場合は、テンプレートベースで生成します

### APIキーの設定（Vercel）

1. Vercelダッシュボードにアクセス
2. プロジェクト → Settings → Environment Variables
3. `OPENAI_API_KEY` を追加して値を設定

### ローカル開発環境

ローカルで開発する場合は、`server`ディレクトリのサーバーを起動します：

```bash
cd server
npm install
npm start
```

その後、`ad-exclusion-request.html`を開いて使用できます。

## セキュリティについて

- ✅ APIキーは環境変数で管理（Vercelの環境変数または`.env`ファイル）
- ✅ バックエンド（Serverless Function）でAPIキーを保護
- ✅ フロントエンドにAPIキーを露出しない

## カスタマイズ

### サンプルデータの変更

`ad-exclusion-app.js` の `SAMPLE_DATA_ARRAY` 配列を編集してください。

### トーンの追加

`ad-exclusion-app.js` と `api/generate-exclusion.js` の `toneSettings` オブジェクトを編集してください。

### デザインの変更

`styles.css` を編集してください（既存のPR TIMESツールと共通のスタイルを使用）。
